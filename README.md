# React Todolist

> 学习React时候的练手项目

后端在这里：[https://github.com/CocaineCong/TodoList](https://github.com/CocaineCong/TodoList)

后续可能会用服务器，方便大家调用接口开发使用。

## 主要涉及到以下几点：
- typescript
- 基于axios封装一个request请求，并且添加拦截器
- 父传值给子组件
- 基于antd和antd pro的一些列组件操作
- 使用redux进行状态的存储

## 功能：

- 用户注册登陆
- 列表的增删改查
